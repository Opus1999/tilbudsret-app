# Tilbudsret App

## App info
- **Navn:** Tilbudsret
- **Koncept:** Scanner danske supermarkedstilbud og genererer ugens billigste middagsretter
- **Sprog:** Dansk (al UI-tekst på dansk)

## Tech stack
- Expo React Native
- TypeScript
- @expo-google-fonts/plus-jakarta-sans

## Design tokens

### Farver
| Navn   | Hex       | Brug                        |
|--------|-----------|-----------------------------|
| Rød    | `#E63946` | Primær accent, CTA-knapper  |
| Grøn   | `#2D6A4F` | Succes, pris-highlights     |
| Creme  | `#FFFDF9` | Baggrund                    |

### Font
- **Plus Jakarta Sans** — bruges til al tekst i appen

## Konventioner
- Al kode og kommentarer på engelsk
- Al UI-tekst på dansk
- Funktionelle komponenter med TypeScript interfaces
- Styles med StyleSheet.create (ikke inline styles)
